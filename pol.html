This file works entirely offline. You're reading it from your phone — no internet needed. Audio uses your phone's built-in Polish voice.

Polish voice is built in. If audio doesn't play, go to: Settings → Accessibility → Spoken Content → Voices → Polish and download a voice.

Go to Settings → General Management → Language → Text-to-Speech and ensure Polish is installed under Google TTS engine.

cz = "ch" in cheese
sz = "sh" in shoe
ż / rz = "zh" like the s in measure
ś / si = soft "sh"
ń / ni = soft "ny"
ą = nasal "on"
ę = nasal "en"
ó / u = "oo" as in food
w = "v" sound
j = "y" as in yes

Use the Slow speed setting when learning a new word. Switch to Normal before using it in conversation.
