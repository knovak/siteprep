# Aus2503 Kangaloon section techdoc addendum

## Hiking topic content model
The Kangaloon section (`decks/aus2503/sections/kangaloon/overview.html`) now includes a **Hiking Near Kangaloon** topic.

### Structure
- One introductory paragraph linking to the general AllTrails Kangaloon search page.
- One bulleted list of grouped nearby hiking regions with:
  - drive-time context,
  - condensed terrain/experience notes,
  - direct AllTrails trail links.
- One quick-selection bullet list that maps trip intent to the best hike area.

### Link preservation rule
All source URLs supplied for this hiking topic are preserved as clickable links in the section content.
