# Baltic Deck Hotel Highlights

This note documents where hotel-location content is stored in the Baltic deck.

- `sections/vilnius/overview.html`
  - Highlight block includes Artagonist Art Hotel and NARUTIS Hotel.
- `sections/riga/overview.html`

- `sections/tallinn/overview.html`

## Content conventions
- Hotel entries are rendered inside `.highlight` blocks.
- Each hotel item includes:
  - Hotel name
  - Brief summary
  - Street address
  - Official hotel website link
  - Google Maps link
