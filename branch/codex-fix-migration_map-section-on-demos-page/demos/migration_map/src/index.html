About the data

This atlas visualizes 48 major migrations and diasporas since 1000 AD. Every number here is an estimate, and many are contested. Pre-1800 figures are order-of-magnitude scholarly estimates; flows drawn with dashed edges carry low confidence. Where sources disagree, the dataset records a range, shown in each migration's detail panel alongside its references.

Two different quantities appear: people moved (arrow width) counts migrants during the period; diaspora today (circle area) is usually a descendant population, which can be far larger — or absent where communities dispersed or assimilated. Some modern flows (Gulf labor, Filipino overseas work) are circular rather than permanent; the notes say so.

This subject includes atrocities — enslavement, expulsion, genocide. The presentation aims to be sober and precise; death tolls in transit are recorded in the detail panels where documented.

Adding new research: the atlas is driven entirely by one JSON file. Drag and drop an updated migrations.json onto this window to load it instantly (it is validated first, with the same rules as the project's test suite). To make a change permanent, edit data/migrations.json in the repository and rebuild.

Keyboard: space = play/pause · ← → = step a year (shift for 10) · Home/End = ends of timeline · + − = zoom · Esc = close panels.
